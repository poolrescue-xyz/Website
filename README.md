# Project Name

A tool to find, claim, and withdraw forgotten or migrated liquidity positions (LPs) built on **BNB Smart Chain (BSC)** and compatible with other EVM networks.

## Technology Stack

- **Blockchain**: BNB Smart Chain + EVM-compatible chains  
- **Smart Contracts**: Solidity ^0.8.20  
- **Frontend**: HTML + ethers.js 
- **Development**: Remix, OpenZeppelin libraries
